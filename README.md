# WallRush 🧱

Онлайн-игра 1 на 1: доведи свой шар до противоположной стороны раньше соперника,
перекрывая ему путь стенами. (Правила — классический «Коридор» / Quoridor.)

**Стек:** Node.js ≥18 + Express + WebSocket (`ws`), фронтенд — vanilla JS (одна страница),
Supabase (аккаунты + рейтинг), хостинг — Railway.

---

## Возможности v1

- **Против ИИ** — средний бот, без таймера, полностью на клиенте
- **Онлайн 1v1** — публичные комнаты (лобби), приватные комнаты по коду, быстрая игра
- **Часы** — банк 5:00 на партию (тикают только в свой ход) + максимум 30 сек на ход,
  красная подсветка на последних 10 секундах
- **Переподключение** — 30 сек на возврат при обрыве связи, иначе поражение
- **Реванш** — по согласию обоих игроков
- **Эмодзи-реакции** — 😂 🫡 🤝 😡
- **Гости** — автоник вида `User9143` (на время вкладки), без сохранения
- **Аккаунты** — email + пароль (Supabase Auth), свой ник, статистика, сброс пароля
- **Рейтинг** — топ-50 по победам, виден всем, попадают только зарегистрированные
- **Языки** — RU / EN

## Запуск локально

```bash
npm install
npm start        # http://localhost:3000
npm test         # самопроверка движка и ИИ
```

Без переменных окружения игра работает в гостевом режиме (без аккаунтов и рейтинга).

## Переменные окружения (`.env.example`)

| Переменная | Описание |
|---|---|
| `PORT` | Порт (Railway ставит сам) |
| `SUPABASE_URL` | `https://ttmfcqmkdfbkweljrbrw.supabase.co` |
| `SUPABASE_ANON_KEY` | anon-ключ проекта (публичный) |
| `SUPABASE_SERVICE_KEY` | service_role ключ (СЕКРЕТ, только на сервере) |

## Деплой на Railway

1. Railway → **New Project → Deploy from GitHub repo** → выбрать `wallrush`.
2. В **Variables** добавить `SUPABASE_URL`, `SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_KEY`
   (service_role ключ: Supabase Dashboard → Settings → API keys → `service_role`).
3. В **Settings → Networking** нажать **Generate Domain** — это адрес игры.

## Настройка Supabase (уже сделано)

- Проект: `wallrush` (`ttmfcqmkdfbkweljrbrw`, регион eu-central-1)

  Dashboard → Authentication → Sign In / Up → Email → выключить **Confirm email**.

## Правила игры

Поле 9×9. У каждого игрока шар и 10 стен. За ход: шаг шаром на соседнюю клетку
**или** одна стена (длина 2 клетки). Через стены ходить нельзя. Если шары рядом —
можно перепрыгнуть через шар соперника (с любой стороны; если прямо за ним стена —
по диагонали). Ставить стену, которая полностью отрезает любому из игроков путь
к его цели, запрещено — система не даст. Победа — первым дойти до
противоположного края.
